\# SQL Server Monitoring Framework



Python-based SQL Server monitoring framework for collecting and storing operational metrics.



\## Features



\- Inventory-driven execution

\- Windows Authentication support

\- SQL Authentication support

\- Long Running Query collection

\- Fragmentation collection

\- Historical repository storage

\- Centralized logging

\- Dynamic database discovery

\- Modular collector architecture
